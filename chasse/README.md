# 🏴‍☠️ La Chasse au Trésor de Corse

Une petite **app web** de chasse au trésor pour les enfants, à jouer autour de la
maison en Corse (piscine, jardin de gravier, cuisine, olivier, terrasse).
100 % hors-ligne, aucun compte, un seul fichier.

## 🎮 Le principe

Les enfants forment **un équipage** et parcourent **5 épreuves** autour de la maison.
Chaque épreuve donne **un chiffre du coffre** ; une fois les 5 chiffres réunis,
l'équipage compose le code et découvre où est caché le **vrai trésor**.

Chaque station propose :
1. **Une énigme** pour trouver le lieu (« là où l'eau est bleue… »).
2. **Une mission par enfant, adaptée à son âge** (grades Mousse / Matelot / Corsaire / Capitaine).
3. **Un mini-jeu dans l'app** : quiz corse, caillou d'or, memory des saveurs,
   message secret à décoder (chiffre de César), cadenas à énigmes.

## 👶 Adapté à chaque âge

| Âge | Grade | Type de mission |
|-----|-------|-----------------|
| 6–7 ans | 🐣 Mousse | Observer, compter, courir |
| 8–10 ans | ⚓ Matelot | Chercher, trouver |
| 11–13 ans | 🗡️ Corsaire | Réfléchir, résoudre |
| 14 ans + | 🧭 Capitaine | Coder, coordonner l'équipage |

## ⚙️ Coin des parents

Depuis l'accueil → **Coin des parents** :
- réglez le **message de fin** = la vraie cachette du trésor (bonbons, pièces chocolat, petit cadeau) ;
- consultez le **code du coffre** ;
- gérez l'équipage et réinitialisez la partie.

Idées de cachette : sous un transat, dans le tas de bois, derrière un pot de fleurs, dans le four éteint.

## 📱 Utilisation

- Ouvrez `chasse/index.html` (ou l'URL `.../chasse/` si hébergé sur GitHub Pages).
- Sur téléphone : **Partager → Sur l'écran d'accueil** pour l'installer comme une app.
- Fonctionne ensuite **hors-ligne**, parfait pour une maison à la campagne. 📶
